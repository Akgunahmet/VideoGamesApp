 ## Features <br>
 * Detailed Game Info from [VideoMusic API](https://rapidapi.com/accujazz/api/rawg-video-games-database) is displayed.
 * Play Pronunciations of Words.
 * Store Last Searched Words in CoreData. <br><br>
